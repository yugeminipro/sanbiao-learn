# 三表入门 · 财务报表学习小程序

> 利润表告诉你赚了多少钱，资产负债表告诉你值多少钱，现金流表告诉你手里有多少钱。

一个专为零基础学习者设计的**交互式财务报表教学工具**，通过可视化展示、实时模拟经营和答题测验，帮助你快速理解利润表、资产负债表、现金流量表三张核心财务报表。

## 功能特色

### 7 个学习页面

| 页面 | 内容 |
|------|------|
| **首页** | 三表关系图、核心等式、形象比喻（视频/照片/水表）、学习路径导航 |
| **利润表** | 完整结构详解 + 毛利率/净利润率 + 流向图 |
| **资产负债表** | 资产方 + 负债权益方完整表格 + 会计恒等式 + 关键指标 |
| **现金流表** | CFO/CFI/CFF 三大类详解 + 企业生命周期模式 + 利润质量分析 |
| **模拟经营** | 8 个滑块实时联动三张报表，智能风险警告 |
| **公司案例** | 6 家真实上市公司 2024 年报数据对比 |
| **答题测验** | 10 道选择题 + 即时反馈 + 得分统计 |

### 模拟经营实验室

拖动参数滑块，实时观察三张报表的联动变化：

- 营业收入、成本率、费用、税率
- 资本支出、新增借款、股利
- 智能检测 6 种风险场景（利润虚胖、现金为负、杠杆过高等）

## 快速开始

本项目是纯前端单文件应用，**无需安装任何依赖**：

1. 下载 `三表学习小程序.html`
2. 用浏览器打开即可使用

或直接访问 GitHub Pages（如已启用）。

## 建议学习路径

```
利润表 → 资产负债表 → 现金流表 → 模拟经营 → 公司案例 → 答题测验
```

## 技术栈

- 纯 HTML + CSS + JavaScript（单文件，零依赖）
- 暗色主题，响应式设计
- CSS Grid / Flexbox 布局
- 原生 JavaScript 实现交互逻辑
