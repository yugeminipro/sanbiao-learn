📋 利润表（Income Statement）

一句话：告诉你公司在一段时间内（比如一年）赚了多少钱。
像一段视频，记录从收入到利润的完整过程。

完整结构（以示例公司为例）

项目金额（万元）
营业收入1,000
减：营业成本-600
= 毛利润400
减：销售费用-80
减：管理费用-50
减：研发费用-30
= 营业利润（EBIT）240
减：利息费用-20
减：所得税-55
= 净利润165

关键概念详解

毛利率 = 毛利润 / 营业收入 = 40%
每卖100块钱的东西，扣除成本后剩下40块。毛利率越高，说明产品越有「附加值」。

营业利润率 = 营业利润 / 营业收入 = 24%
扣除运营费用后的真实赚钱能力。

净利润率 = 净利润 / 营业收入 = 16.5%
最终落到口袋里的比例。

利润表的流向

-

净利润

📊 资产负债表（Balance Sheet）

一句话：告诉你公司在某个时间点（比如12月31日）有多少家底。
像一张照片，定格那个瞬间的财务状况。

完整结构

资产方（公司拥有什么）
项目金额（万元）
流动资产
货币资金（现金）80
应收账款120
存货100
= 流动资产合计300
非流动资产
固定资产（厂房设备）500
无形资产（专利品牌）100
= 非流动资产合计600
总资产900
负债+权益方（钱从哪来）
项目金额（万元）
流动负债
应付账款100
短期借款80
= 流动负债合计180
非流动负债
长期借款220
= 非流动负债合计220
所有者权益
股本200
未分配利润300
= 所有者权益合计500
负债 + 权益900
资产 = 负债 + 所有者权益
900 = 400 + 500 · 永远平衡！

通俗理解

资产：公司拥有或控制的、能带来经济利益的资源。比如现金、设备、专利。
负债：公司欠别人的钱。比如银行贷款、欠供应商的货款。
所有者权益：公司真正属于股东的部分 = 资产 - 负债。也叫「净资产」。

类比：你买了一套100万的房子（资产），其中贷款60万（负债），自己出了40万（权益）。
100万 = 60万 + 40万 ✓

关键指标

流动比率
流动资产/流动负债
= 300/180 = 1.67
＞2 较安全
资产负债率
负债/资产
= 400/900 = 44%
＜60% 较稳健
ROE 净资产收益率
净利润/权益
= 165/500 = 33%
越高越好

💰 现金流量表（Cash Flow Statement）

一句话：告诉你公司在一段时间内现金是怎么流入流出的。
利润可以做假（应计制），但现金不会骗人。现金流是企业的「生命线」。

完整结构

项目金额（万元）
一、经营活动现金流（CFO）
销售商品收到的现金+950
购买商品支付的现金-550
支付给员工的现金-130
支付的税费-60
= 经营现金流净额（CFO）+210
二、投资活动现金流（CFI）
购买设备-100
出售旧设备+20
= 投资现金流净额（CFI）-80
三、筹资活动现金流（CFF）
借入新贷款+50
偿还旧贷款-30
支付股利-40
= 筹资现金流净额（CFF）-20
现金净增加+110

三大类现金流详解

🏭
经营活动 CFO
日常做生意
收到/支付的现金
最重要的指标！
🏗️
投资活动 CFI
买卖固定资产
投资其他公司
通常为负（在扩张）
🏦
筹资活动 CFF
借钱/还钱
发行/回购股票
发分红

企业生命周期与现金流模式

阶段CFOCFICFF特征
🌱 初创期--+烧钱扩张，靠融资活命
🚀 成长期+-+赚钱+借钱一起扩张
🌳 成熟期+--赚钱够花，开始还债分红
🍂 衰退期-++变卖资产+借钱续命

CFO vs 净利润：利润质量分析

CFO > 净利润（210 > 165）：利润质量高！公司赚到的是真金白银，不是纸面富贵。

如果 CFO < 净利润：说明很多利润是「应收」的，钱没收回来，利润只是纸面上的。

如果 CFO < 0 但净利润 > 0：⚠️ 危险信号！公司可能在做假账，或者大量赊销导致现金紧张。
